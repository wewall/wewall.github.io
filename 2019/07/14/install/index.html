SSPanelUimBot 自动化部署教程

Posted on 2019-07-14

访问 @SSPanelUimBot 与机器人交互, 发送 /start 即可注册

发送 /help 即可查看机器人使用帮助, 一般配置前三项即可开始自动部署任务

setup1

设置站点配置

执行后会或得类似的结果

请复制该内容到 Telegram 给机器人发送 /setup1 eyJhc...NzEifQ== 即可完成配置, 此处字符简化删除了一部分, 在配置时必须完善填写.

setup2

设置 Telegram Bot Token

获取方式 @BotFather

获取后给机器人发送 /setup2 770290252:AAEABUpuQl3KKTfMAQ8IEu30EL9djYWyzeU 即可完成配置.

setup3

设置生效群组 ID

将 @ddocbot(Ananke is the best!) 拉入到您的群内, 发送 /id

GID 即为群组 ID 然后给机器人发送 /setup3 -1001492693722 即可完成配置

至此 普通配置结束, 免费版本扣除初装费用后即可开始部署.

setup4

设置支付方式(VIP) 目前支付仅支持当面付, 其他支付后续以投票方式接入

/setup4 f2fpay_app_id:merchant_private_key:alipay_public_key 以 : 间隔

setup5

选择托管区域(VIP)

暂时开放区域 SGP, IRL, US, JP

/setup5 SGP 以空格间隔

setup6

设置自定义URL信息(VIP)

/setup6 suburl:affurl 以 : 间隔

setup7

设置会员群组 ID(VIP)

/setup7 GID GID 即为群组 ID

setup8

会员群准入等级(VIP)

/setup8 3 该示例表示 VIP3 及其以上才可以入群

setup9

开启入群真人检测(VIP)

/setup9 off off 关闭 on开启, 该功能默认关闭
